## Description
This Python project calculates the total investment value of a stock portfolio. Users can enter stock names and quantities, and the program calculates the total investment based on predefined stock prices. The final portfolio summary is also saved to a text file.

## Features
- User inputs stock names and quantities.
- Uses a predefined dictionary for stock prices.
- Calculates total investment.
- Displays portfolio summary.
- Saves the result in a text file.

## Technologies Used
- Python 3
- Dictionary
- Loops
- Input/Output
- File Handling

## How to Run
1. Install Python 3.
2. Open the project folder.
3. Run the following command:

```bash
python stock_portfolio_tracker.py
```
